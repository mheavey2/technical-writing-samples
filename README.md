# Technical Writing Portfolio

👋 Hi, I'm Margaret. I write technical documents for the scientific and tech industries. I also enjoy creating apps and websites.

I create precise, comprehensive technical reports and translating complex technical information into clear, actionable documentation.

## What this portfolio contains

This portfolio contains a variety of samples of my technical documentation.

### Navigating the Portfolio

- To view documentation samples, navigate to `technical-writing-samples/docs`. Documents are grouped based on their type. e.g. tutorials can be found in the `docs/tutorials` directory.
- To view infographic samples, navigate to `technical-writing-samples/infographics`.

## Tools & Technologies

- **Documentation:** Markdown, reStructuredText, Git, GitHub, Canva, Figma, MSOffice, Google Workspace
- **Technical:** JavaScript, TypeScript, HTML5, CSS3, SQL, REST API, ReactJS, NodeJS,ExpressJS, Prisma
